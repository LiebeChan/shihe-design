## 自定义域名配置

部署后到 Vercel 项目设置中添加域名 `yn-ny.com`：

1. 进入项目 → Settings → Domains
2. 添加 `www.yn-ny.com`
3. 按提示修改阿里云 DNS：
   - 记录类型：CNAME
   - 主机记录：www
   - 记录值：cname.vercel-dns.com
